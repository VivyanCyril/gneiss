Balances (gneiss.balances)¶

This module contains modules for calculating balances and creating ETE objects to visualize these balances on a tree.

Functions¶

balance_basis(tree_node) Determines the basis based on bifurcating tree.
balanceplot(balances, tree[, layout, mode]) Plots balances on tree.
